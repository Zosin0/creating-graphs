Graph Concepts

A graph is a data structure that consists of a set of vertices (also called nodes) and a set of edges (also called arcs) connecting these vertices.

Types of Graphs

  • Undirected Graph: A graph where edges have no direction. The edges are represented by unordered pairs of vertices.
  • Directed Graph: A graph where edges have a direction. The edges are represented by ordered pairs of vertices.
  • Weighted Graph: A graph where edges have weights or costs associated with them.

Graph Representation

There are several ways to represent a graph:

  • Adjacency Matrix: A 2D array where the value at index (i, j) represents the presence or absence of an edge between vertices i and j.
  • Adjacency List: A collection of linked lists or arrays where each list/array represents the neighbors of a vertex.

Graph Traversal

Graph traversal refers to the process of visiting all the vertices of a graph. There are two common traversal algorithms:

  • Breadth-First Search (BFS): A traversal algorithm that explores all the vertices of a graph in breadth-first order, i.e., visiting all the neighbors of a vertex before moving on to the next level.
  • Depth-First Search (DFS): A traversal algorithm that explores all the vertices of a graph in depth-first order, i.e., visiting a vertex and then recursively visiting all its unvisited neighbors.
